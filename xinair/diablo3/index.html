What is this tool? This is a quest aggregator using the D3 js API. Learn more about the api at the Blizzard API docs

What is this tool for? It's purpose was to help me and a group of friends find out which quests we still need to complete as a requirement to reset our game. Why? For moar loots, specifically the guaranteed legendary after killing Diablo on freshly reset game! You can read into more detail about the guaranteed legendary drops at incgamer.
